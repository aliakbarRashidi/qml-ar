Detailed Description

Paints image parts not containing markers in black.

This class decorates marker detector and marker mvp provider Adding the ability to detect markers only on predicted area by blackening everything else on the image, except for the area where marker is expected to be

WorldImageCorrespondences assumed to contain marker corners in counter- clockwise order
