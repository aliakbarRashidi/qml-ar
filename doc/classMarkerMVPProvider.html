Detailed Description

This class implements MVPProvider using pose from a detected markers, a calibrated camera's matrix multiplies them and outputs MVP.

INPUT COORDINATE SYSTEM (OpenCV/QML): x right y down z from reader

OUTPUT COORDINATE SYSTEM (OpenGL) x right y up z to reader
