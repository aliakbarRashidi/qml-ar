Detailed Description

This class implements low-pass filter on Pose (Rotation + Translation)

Rotation filtering is done by (q1 + q1).normalize() for quaternions This solution is approximate, see https://stackoverflow.com/questions/21241965/average-transformation-matrix-for-a-list-of-transformations
