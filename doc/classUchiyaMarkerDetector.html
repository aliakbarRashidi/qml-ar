Detailed Description

Wrapper around C-style UchiyaMarkers project obtained from main.cpp file from UchiyaMarkers project.

Notice: some functionality was removed

INPUT/OUTPUT COORDINATE SYSTEM (OpenCV/QML): x right y down z from reader

Config must have additional property 'dots_uchiya' and marker ids must be consecutive numbers from 0 to n - 1

example: { "markers": [ { "name": "big", "id": 0, "x_mm": 9.5, "y_mm": 7.9, "size_mm": 150, "dots_uchiya": [ { "x": 81, "y": 111 }, { "x": 569, "y": 460 } ] } ] }
